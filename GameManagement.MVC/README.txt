- O sistema de gerenciamento de torneio será utiliado para montar as chaves de um torneio 'mata-mata' 
composto por 8 equipes a serem cadastradas no sistema.

- A dificuldade enfrentada foi de idealizar a lógica necessária para o gereciamento de chaves e como trasnportá-lá para
uma view, de forma que o usuário conseguisse dar entrada com a informação do placar do jogo e automaticamente o sistema
atualizaria a chave para a próxima rodada.
